## ğŸ” Smart Contracts
ImmuLink uses **four core smart contracts**:

1. **Manufacturer Approval Contract**
   - Registers manufacturers
   - Uses rating-based approval
   - Prevents unauthorized vaccine production

2. **Trader Registration Contract**
   - Registers distributors/traders
   - Maintains trust scores and ratings
   - Ensures accountability

3. **Cold Vaccine Monitoring Contract (IoT)**
   - Receives real-time temperature data
   - Validates cold-chain compliance
   - Flags violations automatically

4. **Consumer Feedback Contract**
   - Collects post-consumption feedback
   - Updates trust metrics
   - Enhances transparency
